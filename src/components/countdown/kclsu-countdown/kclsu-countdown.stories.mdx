# `<kclsu-countdown>`

<Meta
  title="Other UI/Countdown Clock"
  component="kclsu-countdown"
  argTypes={{
    date: {
      control: 'date',
      description: 'The date to countdown to. Must match date text format.',
    },
    text: {
      control: 'text',
      description: 'The text beneath the countdown',
    },
    msgbg: {
      control: 'boolean',
      description: 'Give the message text a shadowed white block background.',
      table: { defaultValue: { summary: false } },
    },
    textcolour: {
      control: 'color',
      description: 'Optional: set a colour for the text beneath the countdown.',
    }
  }}
/>

### Example

```html
    <kclsu-countdown date='Sepember 20, 2022, 11:00:00' text='Until something happens'></kclsu-countdown>
```


### Build your own
<Canvas withToolbar={true} withSource="open" >
  <Story name="Basic" component="kclsu-countdown" args={{
    date: 'Sepember 20, 2022, 11:00:00',
    text: 'Until something happens'
  }}>
    {args => html`
      <kclsu-countdown
        date="${ifDefined(new Date(args.date) || undefined)}"
        text="${ifDefined(args.text || undefined)}"
      ></kclsu-countdown>`}
  </Story>
</Canvas>

### Appearance

There are four options to customize the button's appearance.
Set the `purple` attribute to use the secondary colour pallete. Set the `rounded` attribute to give the button rounded corners. Set the `small` or `verysmall` attributes to decrease the button's size.

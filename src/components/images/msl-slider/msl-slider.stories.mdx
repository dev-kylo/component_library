# `MSL Image Gallery Slider`
Make sure all images have used the Cloudinary Image Gallery preset.
NOTE: There is currently NO component for this. Use the html snipper below which relies on MSL scripts & styles:

### Necessary styles
gallery-carousel.css

### Necessary scripts
gallery-carousel.js

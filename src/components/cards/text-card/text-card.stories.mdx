# `<text-card>`

### Example

```html
<text-card cardtitle="Card title" subtext="Any sub text" link="/click-through-link"></text-card>
```

### Build your own

<Canvas withSource="open" >
  <Story name="Basic" component="text-card" args={{
    cardtitle: 'Card title',
    link: '/click-through-link',
    subtext: 'Any sub text to go in the card',
  }}>
    {args => html`
      <text-card
        cardtitle="${ifDefined(args.cardtitle || undefined)}"
        link="${ifDefined(args.link || undefined)}"
        subtext="${ifDefined(args.subtext || undefined)}"
      > </text-card>
    `}
  </Story>
</Canvas>

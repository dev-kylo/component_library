# `<kclsu-tabs>`

## Content

### Example

```html
<kclsu-tabs>
  
  <tab-title name="A" active>Title A</tab-title>,
  <tab-area name="A">
    <p>A, from the proto-caananite Aleph, is the first letter in the alphabet</p>
  </tab-area>
  
  <tab-title name="B" active>Title B</tab-title>,
  <tab-area name="B">
    <p>B all the way</p>
  </tab-area>

</kclsu-tabs>
```

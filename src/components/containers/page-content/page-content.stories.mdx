# `<page-content>`

### Copy

```html
<page-content>
    <!-- Add your html inside here -->
</page-content>
```

This is a container for a central column of content. It is the equivalent of the Centre Column Template inside the MSL Skin

### Example

<Canvas withSource="open" >
  <Story name="Basic" component="page-content">
    {args => html`
        <page-content >
            <!-- Add your html here -->
        </page-content>
    `}
  </Story>
</Canvas>


### Using MSL Classes
You can use this html instead of using a component.

```html

<!-- PAGE CONTAINER -->

<div class="container">
    <div class="row">
        <div class="col-xs-12 col-md-8 col-md-push-2">
        <!--START OF PAGE CONTENT -->

            //CONTENT GOES HERE

         <!--END OF PAGE CONTENT -->
        </div>
    </div>
</div>

<!--END OF PAGE CONTAINER -->

```
